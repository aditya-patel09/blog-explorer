# Blog Explorer

Blog Explorer is a modern and responsive blog web application built using React and Tailwind CSS.  
The project allows users to explore blogs related to Artificial Intelligence, Web Development, LLMs, Python, OpenAI, and modern technologies.

The application includes blog searching, domain filtering, tag-based filtering, responsive UI, dynamic routing, and markdown blog rendering.

# Features

- Responsive modern UI
- Search blogs by title or author
- Filter blogs by domain
- Filter blogs using tags
- Dynamic blog detail pages
- Markdown content rendering using ReactMarkdown
- Sticky responsive navbar
- Footer on all pages
- Clean component-based architecture
- Fast performance using useMemo optimization

# Technologies Used

## Frontend

- React.js
- React Router DOM
- Tailwind CSS
- ReactMarkdown
- Vite
